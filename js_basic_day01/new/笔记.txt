1、javascript概述（what、why、when、how）
   理论5分和实操5分
   1、什么是javascript
      简称js，是运行在 javascript解释器 或者 特定引擎中 的解释型脚本语言

      解释型：运行之前不需要检查任何东西，碰到错误，就停止运行，比如：js、php
      编译型：运行之前需要检查语法是否正确，不正确的话就不能运行的，比如：java、c

      javascript解释器：
	1、浏览器自带js解释器
	2、独立安装javascript解释器（node.js）
      支持javascript：PC，平板，手机，电视

   2、javascript的组成：3部分
      核心语法：ECMAScript（ES3/ES5/ES6）	//内功心法
      文档对象模型：Document Object Model(DOM)   //用js动态的操作html和css
      浏览器对象模型：Browser Object Model(BOM)  //操作浏览器的、相对较少

   3、javascript的特点
      1、代码可用任何文本编辑工具编写
      2、无需编译，由javascript引擎解释执行
      3、***弱类型语言
	 由数据 决定 数据类型
	   1	     number
	   "1"	     string
	 强类型语言：是由数据类型 决定 保存的数据是什么	
      4、面向对象的脚本语言（基于原型）	 

   4、javascript的用途
	1、客户端的计算（购物车）
	2、表单的数据验证
	3、浏览器事件的触发和处理（鼠标，键盘，浏览器自动）
	4、网页特效
        5、服务器端异步交互（ajax）

控制台输出：console.log(变量);
查看变量的数据类型：typeof(变量);

   5、js如何在html中运行：
      1、在html页面中嵌套<script>js代码</script>
      2、*工作中，在html中引入js文件
	<script src="xx.js" type="text/javascript" charset="utf-8">
		//引入文件后，此位置不可以在写任何内容
        </script>

   1、JS调试
      1、代码运行期间，如果碰到错误，解释器会终止本次代码块，但是不影响后续代码块
      2、如果程序出现逻辑错误
	 逻辑错误，并非语法错误，逻辑错误不会报错
	   打桩输出：console.log();
	 何时使用：如果你发现自己没有报错，但是出来的结果跟你想象中有区别

   2、JS语法规范
      1、语句
	 被javascript解释器所执行的代码，就是语句
	 由 表达式、关键字、运算符 ...组成
   
	 语句是严格区分大小写的
	 console.log(1);
	 Console.log(2);

   3、变量 和 常量
      1、*变量

	 硬盘：存储数据，文件（外部存储器）
	 CPU：中央处理器（计算的）
	 内存：是[临时保存]程序在运行过程中，所需用到的数据
	       这是一段临时的数据

	       内存空间：保存数据的一个地方
	       内存地址：内存空间的门牌号(标识)：0x000000000000000dsae321dsa
			 如果内存地址过长的话，可以给内存地址起一个别名，方便记忆

	 
	 1、什么是变量
	    变量就是内存中的一段内存空间，主要目的就是保存数据
	   *也可以理解成是一个 可以变化的量(数据)

	  想要找到变量的话，可以通过内存地址，但是内存地址又太麻烦了，所以起一个变量名

	 2、变量的声明
	    1、通过关键字 var 声明(定义、创建)变量
	       语法：var 变量名;
		     var name;
		     var age;
		     var gender;

	    2、赋值：通过 = 完成对变量的赋值
	       =：赋值符号

	       语法：变量名=变量值 -> 赋值表达式
		     var age;
		     age=200;
		     age=201;

	    3、注意：如果只声明变量，但是未赋值，会被默认赋值为 undefined！

	    问题：在程序中声明5个变量，分别保存姓名，性别，年龄，身高，体重？？？
		var name;
		var age;
	        var gender;
		var height;
		var weight;

	    解决：一句话直接声明多个变量
		var name,age,gender,height,weight;

	    注意：在声明变量时候，是可以省略var的，
		  但是绝对不建议这样做，有可能造成全局污染
		  一般情况不要省略var，但是后面我们可能会遇到

	3、变量名 命名规范
	   1、不能使用JS的关键词和保留关键字进行命名
	   2、不能以数字开头
	   3、尽量见名只意，百度翻译
	      var name;//推荐
	      var xingming;//尽可能别用

	      //可能当时用的时候还记得
	      var a;
	      var b;
	      var c;//千万别用
	   4、可以使用驼峰命名法，下划线命名法
	      下划线：index_header index_banner index_news index_about
	      驼峰：indexHeader indexBanner

	4、声明变量时，可以直接赋值（初始化）
	   var 变量名=变量值;

	   初始化：声明变量，但是并未赋值，默认值为undefined

        5、变量的使用：
	   在你使用变量名的时候其实就是在使用变量的值

	总结：1、var 变量名=变量值;
	      2、console.log(变量名);
	      3、如果这个变量未声明，会报错

   4、运算符：
      1、算术运算符：+ - * / %

	 +：如果两边的操作数都是数字，那么就是相加
	    如果只要有一边出现了字符串，那么加运算就变为了字符串拼接

	 var num1=15;
	 var num2=18;
	 num1+num2;//33

	 var name="张三丰";
	 var age=93;
	 name+age;//"张三丰93"

         %：取余：取两个操作数运算结果的余数
	    俗称：模

	 5%2; //1
	 2%5; //2

	 作用：
	   1、获取某数字的最后n位
		1234 % 10 //4
	   2、算奇偶性
		num % 2 //看结果如果是1，则为奇数，如果是0则为偶数
