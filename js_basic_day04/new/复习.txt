1、循环
   3要素：

   循环变量;
   while(循环条件){
	循环体;
	变量自增;
   }

   for(循环变量;循环条件;变量自增){
	循环体;
   }

2、数组
   创建数组：
     1、*直接量方式：var arr=[1,2,3,4,5];
     2、构造函数方式：var arr=new Array(1,2,3,4,5);

   获取数组的元素
     arr[下标];//下标从0开始，到arr.length-1结束

   添加 或 修改元素
     arr[下标]=新值;
     arr[arr.length]=新值;//永远是往数组的最后一个添加

   如果希望对数组的每个元素执行相同 或者 相似的操作
     for(var i=0;i<arr.length;i++){
	arr[i];//当前元素
     }
